This page demonstrates: Performing redirection of a page request.
Comments: The name of the page to be redirected to is defined by an ENUM variable

Possible redirection to a page with an error message, stating that the range has been violated (>100)

testword = :="Data".testword:

The AWP special variables 'HEADER:status' and 'HEADER:Location have to be defined with an AWP_Out_Variable statement within this page, to request automatic transfer of a PLC-Variable into the corresponding HTTP header variable for the HTTP response.
An ENUM is used here to automatically set a defined string for a value (definition see _enumdefs.htm or Web2PLC project settings). 

<!-- AWP_Out_Variable Name='"Data".headerStatus' -->
<!-- AWP_Out_Variable Name='"Data".headerLocation' Enum="redir_enum" -->
