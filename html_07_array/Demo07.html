This page demonstrates: Expanding all elements of an array using AWP_Start_Array.

With the AWP statement "AWP_Start_Array" the lines until "AWP_End_Array" are automatically repeated for each array element. The predefined variables "ArrayIndex" and "value" are replaced by the actual index and the value for each array element : 
    <!-- AWP_Start_Array Name="<arrayname>" -->
    <tr><td>:=ArrayIndex:</td><td>:=value:</td></tr>
    <!-- AWP_End_Array -->
